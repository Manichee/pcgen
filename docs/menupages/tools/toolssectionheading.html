Tools Menu Options

The Tools Menu contains several tools available for use in and with PCGen.

The Preferences menu option allows the user to configure portions of PCGen's user experience and rules handling.

GMGen is PCGen's game master aid, providing a number tools to assist the GM in tracking initiative, encounters, experience tracking and several other tools.

The Log & Memory Use menu option opens the Log window where errors are logged during program.

The Logging Level menu option allows the user to set the level of errors to be logged.

The Debug Console menu option opens the debug window.
